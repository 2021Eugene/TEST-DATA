Задание
Напишите набор команд с комментарием их работы которые вы примените для решения следующих задач:

1)Выведите список всех файлов в папке html

	ls -a /var/www/html 
	# Просмотр содержимого каталога html с помощью команды ls 
         -a (all) — выводит список всех файлов каталога, в том числе и скрытые файлы. .. и . в верхней части списка относятся, соответственно, к каталогу на уровень выше текущего и текущему.

2)Найдите в файле access.log все упоминания строки "404"

	$ grep -w 404 /var/logs/access.log
	   # Команда grep (расшифровывается как global regular expression print). Она даёт возможность пользователям сортировать и фильтровать текст на основе сложных правил.
	     Утилита grep решает множество задач, в основном она используется для поиска строк, соответствующих строке в тексте или содержимому файлов.
	     Также она может находить по шаблону или регулярным выражениям.
	     -w - искать шаблон как слово, окружённое пробелами;

3)На основании файла 404.html сделайте файл 403.html заменив в его содержимом все упоминания 404 на 403

	cd ~/var/www/html/
	   # Перехожу в папку html
	cp 404.html 403.html
	   #копирую файл 404.txt в новый файл 403.html
	$ sed 's/404/403' ./403.html 
	   #заменяю в файле 403.html все упоминания 404 на 403

4)Модифицируйте права доступа и владельца файла index.html следующим образом: Текущий пользователь должен иметь права 
на чтение/запись Группа www-data на чтение Все остальные не имеют доступа к файлу

	cd ~/var/www/html/
	   # Перехожу в папку html
	$ chown Текущий пользователь:www-data .index.html
	   #заменяем владельца и группу файла index.html, на текущего пользователя и группу  www-data
	$ chmod u=rw index.html
	   #Устанавливаю текущему пользователю права на чтение/запись вместо имеющихся
	$ chmod g=r index.html
	   #Устанавливаю группе www-data права на чтение вместо имеющихся
	$  chmod o-rwx index.html
	   #забираю все права у остальных пользователей
	
5)Сделайте символическую ссылку на файл access.log в папке html.

	cd ~/var/www/html/
	   # Перехожу в папку html
	$ ln -s access.log softlink
	   #создаю символическую ссылку Linux, для этого используется команда ln с опцией -s

6)Выведите на экран все файлы в папке var

	ls -R /var
	   #Просматриваю список файлов с помощью команды (ls -R)
	    -R — recursive (рекурсивный). Этот параметр также показывает содержимое всех каталогов, вложенных в текущий. 	

7)Создайте новый файл 503.html в папке html с следующим содержимым:
<html>
    <head>
        <title>503</title>
    </head>
</html>

	cd ~/var/www/html/
	   # Перехожу в папку html
	$ echo "<html>
                    <head>
                        <title>503</title>
                    </head>
                </html>" > 503.html
	   #создаю новый файл 503.html
